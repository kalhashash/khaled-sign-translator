## الامتدادات المدعومة
`png` · `jpg` · `jpeg` · `webp` · `gif`

مثال: `alef.png` أو `beh.jpg`.

## تخصيص الأسماء
إذا كانت صورك بأسماء مختلفة، عدّل قسم `letters` في `mapping.json` ليطابق ملفاتك.
